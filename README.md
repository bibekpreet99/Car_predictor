# Car Predictor

### Overview
This website uses a trained car predictor model to predcit the price of the car using Random Forest classifier
Stack Used
1. Flask
2. OpenCV
3. HTML/CSS

### Live Link
https://car-pr3dictor.herokuapp.com/

### Novelty
1. Simple and easy User Interface.
